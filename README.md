## Описание программы

Book Tracker — это графическое приложение на Python для ведения личного списка прочитанных книг.  
Программа позволяет добавлять книги, удалять их, фильтровать по жанру и количеству страниц, а также сохранять и загружать данные в формате JSON.

### Основные возможности

- Добавление книги с указанием:
  - названия
  - автора
  - жанра
  - количества страниц
- Валидация ввода (пустые поля, числовые значения)
- Фильтрация:
  - по жанру
  - по количеству страниц (больше заданного значения)
- Удаление книги двойным кликом
- Сохранение данных в JSON
- Загрузка данных из JSON
- Статистика по книгам (всего книг, страниц, среднее, популярный жанр)
- Автоматическая запись даты добавления
