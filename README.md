# Генератор цитат — Quote Generator

**Автор:** Николай Сорокин  
**Файл:** Main.py  
**Версия:** 2.0

---

## Цель проекта

Создание GUI-приложения для генерации случайных цитат с возможностью просмотра истории, фильтрации по автору и теме, а также сохранением истории в JSON-файл.

---

## Функциональные возможности

- ✅ Генерация случайной цитаты (кнопка «Сгенерировать цитату»)
- ✅ Отображение цитаты на экране (текст, автор, тема)
- ✅ История всех сгенерированных цитат в виде таблицы
- ✅ Фильтрация истории по **автору** и по **теме**
- ✅ Сохранение истории в `quotes_history.json`
- ✅ Очистка истории
- ✅ Статистика (общее количество цитат, количество уникальных авторов)
